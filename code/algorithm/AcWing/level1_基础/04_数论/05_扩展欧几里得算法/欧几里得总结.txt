因为裴蜀定理:对于任意正整数 a, b, 一定会存在非零整数 x,y, 使得 ax + by = gcd(a,b) => ax + by = c
对于 ax + by = gcd(a,b) 这样的不定方程, 可以用 "扩展欧几里得" 算法求出
    扩展欧几里得:可以用于求解方程 ax + by = gcd(a,b) 的解 
        - 当开始迭代时将系数代入可以推得: x = y', y = x' - ⌊a/b⌋ * y'
    

对于求解一般的方程 ax + by = c (一般方程是利用欧几里得求出的)
设: d = gcd(a,b), 这样的不定方程有解情况为: d|c
        - 因为 a 和 b 都是最大公因数的倍数, 所以累加在一起(ax+by)依旧是最大公约数的倍数
          如果 d|c 那么 c 也是最大公因数倍数, 如果不能整除那一定无解
  求解方法: 先用扩展欧几里得求出 ax₀ + by₀ = d 的解, 但是我们要求的是 c
            则 a(x₀ * c/d) + b(y₀ * c/d) = c
            故而特解为 x' = x₀ * c/d, y' = y₀ * c/d
            而 通解 = 特解 + 其次解
            而其次解即为方程 ax + by = 0
            x = x'+k 

求 线性同余方程, 因为扩展欧几里得算法对 ax + my = (a, m) = d
